# QURAN-EMBED
Embed Quran Lengkap dengan Terjemah Dua Bahasa dan tafsir Kemenah RI. Anda bisa download package, lalu upload ke server Anda.

# Install
Download file atau pakai git, lalu upload ke server anda. Itu saja.
